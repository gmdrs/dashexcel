# Dashboard de Vendas no Excel

![Excel](https://img.shields.io/badge/Microsoft_Excel-217346?style=for-the-badge&logo=microsoft-excel&logoColor=white)
![Data Analysis](https://img.shields.io/badge/Data_Analysis-000000?style=for-the-badge&logo=google-analytics&logoColor=white)

Este repositório contém um dashboard interativo de vendas desenvolvido inteiramente no *Microsoft Excel*. O painel foi construído utilizando Tabelas Dinâmicas e Gráficos Dinâmicos, 
permitindo uma análise rápida e visual do desempenho comercial da equipe de vendas ao longo de diferentes anos.

## Conteúdo do Repositório

- **`dashboard_vendas.xlsx`**: Arquivo original do Excel contendo a base de dados bruta, as abas de apoio com as tabelas dinâmicas para cálculos e o dashboard interativo.
- **`dashboard vendas excel.mp4`**: Um vídeo simples demonstrando o dashboard em funcionamento, ilustrando a interatividade dos segmentadores de dados (filtros).

## Funcionalidades e Métricas Inclusas

O dashboard apresenta três visões principais que se atualizam dinamicamente conforme os filtros aplicados:

*   **Valor Médio:** Gráfico de barras horizontais que compara o ticket médio/valor médio gerado por cada vendedor.
*   **Vendas (Evolução Temporal):** Gráfico de linhas demonstrando a performance de vendas ao longo dos meses (1 a 12), permitindo a comparação de tendências.
*   **Vendas Perdidas:** Gráfico de colunas agrupadas ilustrando o volume de oportunidades/vendas perdidas mês a mês.
*   **Filtros Dinâmicos (Segmentadores de Dados):** 
    *   Segmentação por **Ano** (2021, 2022, 2023).
    *   Segmentação por **Vendedor** (Adriano, Arthur, Bruno, Everton, Filipe, Gabriel, Jorge, Sávio). É possível selecionar múltiplos vendedores simultaneamente para cruzar informações.


## 🛠️ Tecnologias e Técnicas Utilizadas

*   **Microsoft Excel** — Ferramenta principal de desenvolvimento e estruturação.
*   **Tabelas Dinâmicas** — Para agregação e sumarização rápida dos dados de vendas, visitas e fechamentos.
*   **Gráficos Dinâmicos** — Para a visualização interativa conectada à base.
*   **Segmentação de Dados** — Para criar uma experiência fluida de painel, permitindo que o usuário final filtre as métricas com apenas um clique.
